## About
This repository was created to share codes created for Outvio tasks

## Construction

This is all developed in Jupyter Notebooks using Python and Pandas

## Installation & Dependencies

- python
- jupyternotebooks
- openpyxl
- pandas
- numpy

 - First you need Python installed in your machine, you can download it directly on https://www.python.org/downloads/

 - Check the version of python that you're using and install pandas using pip3 (or pip).
 - use "pip3 install pandas" 
 - use "pip3 install jupyter"
 - use "pip3 install numpy"
